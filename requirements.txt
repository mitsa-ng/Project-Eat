# ── AI English Essay Corrector — requirements ─────────────────────────────────
# Python 3.10+  |  Windows 10  |  RTX 2080 Ti (CUDA 11.x)

# ── Step 1: Install CUDA-enabled PyTorch FIRST (required for GPU EasyOCR) ─────
# Run this BEFORE pip install -r requirements.txt:
#
#   pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
#
# Verify CUDA is working:
#   python -c "import torch; print(torch.cuda.is_available(), torch.cuda.get_device_name(0))"
# Expected output:  True  NVIDIA GeForce RTX 2080 Ti
#
# If you installed regular torch (CPU-only) by mistake, uninstall first:
#   pip uninstall torch torchvision torchaudio -y
# Then re-run the cu118 install above.

# ── Step 2: Install remaining dependencies ────────────────────────────────────
