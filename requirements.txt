# ── AI English Essay Corrector — requirements ─────────────────────────────────
# Python 3.10+  |  Windows / Linux / macOS

# ── Step 1: Install PyTorch FIRST (required for GPU EasyOCR) ──────────────────
# Windows/Linux with NVIDIA GPU — install the CUDA wheel BEFORE this file:
#
#   pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
#
# Verify CUDA is working:
#   python -c "import torch; print(torch.cuda.is_available())"
#
# macOS (Apple Silicon) — the standard wheel includes MPS, auto-detected:
#
#   pip install torch torchvision
#   python -c "import torch; print(torch.backends.mps.is_available())"
#
# If you installed regular torch (CPU-only) on a CUDA machine by mistake:
#   pip uninstall torch torchvision torchaudio -y
# Then re-run the cu118 install above.
